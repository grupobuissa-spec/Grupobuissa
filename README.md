# Grupo Buissa | Industria y Agro

Sitio web oficial de **Grupo Buissa**, empresa especializada en soluciones para la industria y el agro.  
Este repositorio contiene el código fuente del sitio publicado en **GitHub Pages**.

---

## 🚀 Características
- Catálogo de productos industriales y agro.
- Diseño responsive adaptado a móviles y escritorio.
- Integración con WhatsApp para cotizaciones rápidas.
- Formularios de contacto y pedidos.
